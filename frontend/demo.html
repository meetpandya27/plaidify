How it works

📋

1. Blueprint

A JSON file defines how to authenticate and where to find data on a site.

🌐

2. Browser

Playwright launches a real Chromium browser and executes the auth flow.

🔍

3. Extract

CSS selectors pull data from the page. Transforms normalize types.

📦

4. Return

Clean, structured JSON is returned via the REST API. Ready to use.
